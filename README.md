# NeoStore

NeoStore é um projeto de e-commerce fictício desenvolvido para portfólio, com foco em interface moderna, experiência visual premium e navegação fluida.

## Recursos
- Página inicial com carrossel e destaques
- Catálogo de produtos
- Página de produto dinâmica por parâmetro na URL
- Carrinho lateral com controle de quantidade
- Toast ao adicionar produto
- Layout responsivo
- Estilo moderno com Tailwind CSS

## Tecnologias
- HTML5
- Tailwind CSS
- JavaScript

## Objetivo
Este projeto foi criado para demonstrar organização de frontend, construção de interface para ecommerce e integração de interações com JavaScript puro.
